Element

Element refers to the tag for which we want to apply style i.e. <p>. This style is applied to everytime we use the specific tag!

Class

In this case, we can build a class with a specif style which can be apply to any tag when is specified so. i.e. <p class="myclass">.

this paragraph is using "myclass" style, this can be defined within <head> tag or in the ".css" file.

ID Selector

We can also define an id name as it is defined in the <head> and it can be used as: <div id="myid">

this "div" uses the id style "myid"
Keep in mind that "id" can only be applied to one element.

Grouping selectors

We can also group several selectors which have the same style as follows:

selector1, selector2 = { color = green; }
